## Usage Guide

### Getting Started
1. Open the website
2. Click "Connect Account" to authenticate with Pollinations.ai
3. Create personas for both user and AI
4. Start chatting!

### Creating Personas
- **User Personas**: Define who you want to be in the conversation
- **AI Personas**: Define the AI's character, backstory, and response style
- Each persona pair has its own chat history and memory

### Memory System
- **Short-Term Memory**: Recent conversations (automatically managed)
- **Long-Term Memory**: Important facts and details (can be manually added)
- **Keywords**: Auto-extracted and manually added keywords for memory retrieval

### Image Generation
1. Click the image button to switch to image mode
2. Enter your prompt
3. Optionally use "Enhance" to improve the prompt
4. Select a model and generate

### Model Selection
- Text and image models are fetched dynamically from Pollinations.ai
- Pricing information is displayed for each model
- Models can be filtered based on capabilities

## Privacy & Security

- All data is stored locally in your browser (localStorage)
- No data is sent to our servers
- API calls go directly to Pollinations.ai
- BYOP model means you control your API keys and costs
